CompArt

Richard Johnston

Description of this course:

In this course you will learn the how basic properties of computer science and apply them in new and artistic ways. This class is an introductory computer science course that aims to teach simple properties of computer science by working in the language of processing on artistic projects while also teaching essential parts on computer science such as recursion and object oriented coding.
This class will teach you the interworkings of the complex art made my modern day computerized artists.
The language this class is done in is processing which is a program originally written in java, but cleanly polished to make it on of the most comprehensive computerized art interfaces.
